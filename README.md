## ‚ú® About

A small React + Vite demo app for generating QR codes with live customization (size, colors, error correction, styles, optional logo) and exporting the result.

> [!NOTE]  
> This App was completly generated by **Gemini 3 Pro**

## ‚≠ê Features

- Generate QR codes from text or URLs
- Customize output:
	- Size and quiet zone
	- Foreground/background colors
	- Error correction level
	- QR style (squares / dots / fluid)
	- Optional logo (URL, width/height, opacity, padding, padding style)
	- Option to remove QR modules behind the logo
- Download the generated QR code as a PNG
